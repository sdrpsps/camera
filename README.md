# Electron Camera

基于 Electron + Vue3 + Vite + Pinia 开发的桌面摄像头软件，方便直播录课、短视频拍摄等场景。[下载地址](https://github.com/sdrpsps/electron-camera/releases)

## 功能特点

- 支持多摄像头环境下选择特定摄像头

- 窗口本体始终置顶

- 窗口本体显示形状修改

- 支持自定义边框宽度

- 支持自定义边框颜色

- 跨平台支持（ macOS、Windows ）
